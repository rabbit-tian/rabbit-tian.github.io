1. 例题1

    ```
    // 同步任务
    console.log(1)
    // 异步任务，要挂起
    setTimeout(function () {
        console.log(2)
    },0)
    // 同步任务
    console.log(3)
    ```
    - 结果： 1，3，2
    - 分析：先处理同步任务，再处理异步任务

2. 例题2
    
    ```
    console.log(1)
    while(true) {
        
    }
    console.log(2)
    ```
    - 结果： 1
    - 解析：都是同步任务，while没有执行完，结果只有1

3. 例题3
    
    ```
    console.log(1)
    setTimeout(function () {
        console.log(2)
    },0)
    while(true) {
        
    }
    ```
    - 结果： 1
    - 解析：先执行同步任务 ，while一直没有执行完，2就不会被打印出来

4. 例题4
    
    ```
    for (var i = 0;i<4;i++) {
        setTimeout(function () {
            console.log(i)
        },1000)
    }
    ```
    - 结果： 3
    - 解析：先执行同步任务 for循环，i=3，再执行异步任务定时器，结果为3
