### let 和 const
1. for  循环中使用let 形成块级作用域
2. es6 实现块级作用域：改变变量的名字
3. const: 命名后，不能改变

### 解构赋值
1. `let {a,b,c} = {1,2,3}`
2. `let {d,e} = {d:1,e:2}`
3. 深度解构
    
    ```
    let obj = {
        p: [
            "hello",
            {y:"world"}
        ]
    }
    
    let = { p : [ x,{ y }] } = obj
    ```

4. 应用
    
    ```
    function add ([x,y]) {
        return x + y
    }
    add([1,2])
        
    function add({
        name,
        list: [x,y]
    }) {
        return name + x
    }
    
    add ({
        name: 'tian',
        list: [11,12]
    }) 
    ```
5. 字符串解构
    - `const [a,b,c,d,e] = 'hello'`

6. rest 解构
    - const {p, ...rest} = {p:1,a:2,c:3}
    - p = 1, rest 去除p的剩余值
