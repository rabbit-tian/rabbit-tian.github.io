### 创建对象有几种方法
1. let obj1 = {a:1}（字面量方法）
2. let obj2 = new Object({a:1})
3. 构造函数的方式

    ```
    let M = function (num) {this.a = num}
    let obj3 = new M(1)
    ```
4. Object.create 的方法

    ```
    let num = {a:1}
    let obj4 = Object.create(num)
    ```
